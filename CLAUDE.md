# AI 課業輔助與作業草稿生成系統

## Tech Stack
- **Backend**: Python FastAPI + SQLAlchemy + SQLite (async)
- **Frontend**: React + Vite + Tailwind CSS v4 + TypeScript
- **AI**: OpenAI-compatible API (OpenRouter)
- **Deployment**: Single container, single port (8000)

## Project Structure
```
backend/         FastAPI backend (also serves frontend static files)
  app/
    main.py      App entry point (API + SPA serving)
    config.py    Environment config
    database.py  SQLAlchemy async setup
    models/      SQLAlchemy models
    routers/     API routes (auth, tasks, history, admin)
    services/    Business logic (auth, file_parser, ai, progress, export, task)
    utils/       Security, validators, file_utils
  static/        Built frontend (generated by build.sh)
frontend/        React frontend source
  src/
    api/         API client (axios)
    components/  Reusable UI components
    pages/       Route pages
    types/       TypeScript types
```

## Development
```bash
# Backend only (with hot reload)
cd backend && uvicorn app.main:app --reload --port 8000

# Frontend dev (with proxy to backend :8000)
cd frontend && npm run dev

# Build & run all-in-one on port 8000
./build.sh
cd backend && uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## Environment Variables
Copy `backend/.env.example` to `backend/.env` and configure.

## Docker
```bash
docker compose up --build    # Single container on port 8000
```
